How to Use

This header component is designed to be easily customizable and reusable. Simply include the CSS and JavaScript files, then initialize the component with your preferred configuration.

// Basic initialization
document.getElementById('headerContainer').innerHTML = HeaderComponent.createHeader();

// Custom configuration
const customConfig = {
    logo: {
        text: "Your Brand",
        url: "/",
        showIcon: true
    },
    navigation: [
        { text: "Home", url: "/", active: true },
        { text: "About", url: "/about", active: false },
        { text: "Services", url: "/services", active: false }
    ],
    cta: {
        text: "Contact Us",
        url: "/contact",
        style: "primary"
    }
};

document.getElementById('headerContainer').innerHTML = 
    HeaderComponent.createHeader(customConfig);

Features Overview

The header component includes several advanced features:

  • Scroll Effects: Header hides on scroll down, shows on scroll up
  • Backdrop Blur: Modern glass-morphism effect with backdrop filtering
  • Mobile Menu: Animated hamburger menu with smooth transitions
  • Social Links: Configurable social media icons
  • Search Overlay: Full-screen search with suggestions
  • Theme Toggle: Dark/light mode with icon animations
  • Accessibility: Keyboard navigation and screen reader support
  • Performance: Optimized animations and efficient event handling

Customization

The component uses CSS custom properties (variables) for easy theming. You can override these variables to match your brand colors and spacing preferences.

/* Custom theme variables */
:root {
    --accent-primary: #your-primary-color;
    --accent-secondary: #your-secondary-color;
    --header-height: 90px;
    --nav-gap: 2.5rem;
    /* ... more variables */
}
