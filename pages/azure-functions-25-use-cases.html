Retail — Invoice & Receipt OCR Ingestion Pipeline

Industry: Retail / Finance Operations  ·  File Processing

1. Business Context & Problem Statement

A retail chain with 1,200 stores receives ~80,000 supplier invoices per month in mixed formats (PDF, JPG, scanned). Finance clerks manually key data into SAP — ~6 min/invoice, ~$1.4M/year labor cost, with 14-day average DSO inflation due to processing delays and a 7% data-entry error rate.

2. Why Azure Functions?

  • Blob trigger fires the instant a document lands — no polling overhead
  • Pay-per-execution: month-end spike is 8× daily average — App Service or AKS would idle 70% of the time
  • Auto-scale to 200 instances during peaks
  • vs Logic Apps: 3–5× cheaper at this scale; better retry/error semantics
  • vs AKS: no cluster ops, no node patching, no HPA tuning

3. High-Level Architecture

[Email / SFTP / Scanner] -> [Blob Storage: /incoming] v (BlobTrigger) [Azure Function: ExtractInvoice] v Azure AI Document Intelligence (prebuilt-invoice) v [Durable Orchestrator: Validate & Enrich] v [Service Bus: invoices-validated] v [Function: PostToSAP via OData] v [Cosmos DB: audit trail] [App Insights + Log Analytics + Power BI]

4. Step-by-Step Implementation

  1. Requirement breakdown: extract vendor, PO#, line items, totals, VAT; confidence >= 0.85 -> auto-post; else route to human queue.
  2. Provision resources: Function App (Premium EP1 for VNet -> SAP), ADLS Gen2, Document Intelligence, Service Bus, Cosmos DB, Key Vault.
  3. Runtime: C# .NET 8 isolated worker (best SDK + perf).
  4. Triggers/bindings: BlobTrigger on incoming/{name}, ServiceBus output binding, Cosmos output binding for audit.
  5. Core logic: call Document Intelligence -> map fields -> validate against vendor master -> enqueue.
  6. Integrate: SAP via OData with managed identity, Key Vault for credentials, Power BI dataset refresh on completion.
  7. Deploy: Bicep + GitHub Actions, slot swap for zero downtime.
  8. Monitor: App Insights custom metric InvoiceConfidence; alert on spike of low-confidence docs.
  9. Optimize: exponential retry (5 attempts), poison queue at > 5 failures, Premium pre-warmed instances = 3.

5. Code Snippet (C# .NET 8 isolated)

[Function("ExtractInvoice")]
[ServiceBusOutput("invoices-validated", Connection = "SbConn")]
public async Task<InvoiceMessage> Run(
    [BlobTrigger("incoming/{name}", Connection = "StorageConn")] Stream blob,
    string name,
    FunctionContext ctx)
{
    var log = ctx.GetLogger<ExtractInvoice>();
    var client = new DocumentAnalysisClient(
        new Uri(_cfg["DI_ENDPOINT"]),
        new DefaultAzureCredential());

    var op = await client.AnalyzeDocumentAsync(
        WaitUntil.Completed, "prebuilt-invoice", blob);
    var doc = op.Value.Documents.FirstOrDefault()
        ?? throw new InvalidOperationException("No invoice detected");

    var msg = new InvoiceMessage {
        BlobName       = name,
        VendorName     = doc.Fields["VendorName"]?.Value?.AsString(),
        InvoiceTotal   = doc.Fields["InvoiceTotal"]?.Value?.AsCurrency().Amount ?? 0,
        Confidence     = doc.Confidence,
        RequiresReview = doc.Confidence < 0.85
    };
    log.LogInformation("Extracted {Vendor} conf={Conf}", msg.VendorName, msg.Confidence);
    return msg;
}

[Function("ScoreTransaction")]
[EventHubOutput("tx-decisions", Connection = "EhOut")]
public async Task<Decision[]> Run(
    [EventHubTrigger("tx-stream", Connection = "EhIn")] EventData[] events)
{
    var decisions = new List<Decision>(events.Length);
    foreach (var e in events)
    {
        var tx       = JsonSerializer.Deserialize<Transaction>(e.EventBody);
        var profile  = await _redis.HashGetAllAsync($"cust:{tx.CustomerId}");
        var features = FeatureBuilder.Build(tx, profile);
        var score    = await _mlClient.ScoreAsync(features); // <30ms p99
        var verdict  = score > 0.82 ? "BLOCK" : score > 0.55 ? "STEP_UP" : "ALLOW";
        decisions.Add(new Decision(tx.Id, verdict, score));
    }
    return decisions.ToArray();
}

[Function("TransformHL7")]
public async Task Run(

    await _ahds.PutBundleAsync(bundle);

    await actions.CompleteMessageAsync(msg);
}

5. Code Snippet (Python)

import azure.functions as func
import numpy as np, json
from scipy.fft import rfft

def main(events: list[func.EventHubEvent]) -> str:
    rows = []
    for e in events:
        d = json.loads(e.get_body().decode())
        vib = np.array(d["vibration"])           # 100 samples
        fft_bins = np.abs(rfft(vib))[:10]
        rows.append({
            "assetId":  d["assetId"],
            "ts":       d["ts"],
            "rms":      float(np.sqrt(np.mean(vib**2))),
            "kurtosis": float(((vib - vib.mean())**4).mean() / vib.var()**2),
            "fft":      fft_bins.tolist(),
            "tempC":    d["temp"]
        })
    return json.dumps(rows)   # output binding -> ADX via Event Hub

const { app, output } = require('@azure/functions');
const sbOut = output.serviceBusQueue({ queueName: 'shipment-events', connection: 'Sb' });

app.http('AcceptWebhook', {
  methods: ['POST'], authLevel: 'anonymous',
  extraOutputs: [sbOut],
  handler: async (req, ctx) => {
    const carrier = req.params.carrier;
    const raw = await req.text();
    if (!verifyHmac(carrier, raw, req.headers.get('x-signature'))) {
      return { status: 401 };
    }
    ctx.extraOutputs.set(sbOut, {
      carrier, payload: JSON.parse(raw), receivedAt: Date.now()
    });
    return { status: 202 };
  }
});

[Function(nameof(ClaimReview))]
public async Task<ClaimResult> ClaimReview(
    [OrchestrationTrigger] TaskOrchestrationContext ctx)
{
    var claim = ctx.GetInput<Claim>()!;

    var docs = await ctx.CallActivityAsync<List<DocFacts>>(
        "ExtractText", claim.Files);

    var decision = await ctx.CallActivityAsync<Decision>(
        "AutoAdjudicate", docs);

    if (decision.Confidence < 0.9)
    {
        await ctx.CallActivityAsync("NotifyAdjuster", claim.Id);
        using var cts = new CancellationTokenSource();
        var approval = await ctx.WaitForExternalEvent<Approval>(
            "AdjusterDecision", TimeSpan.FromDays(2), cts.Token);
        decision = decision with { Outcome = approval.Outcome };
    }
    await ctx.CallActivityAsync("NotifyAndPay", decision);
    return new ClaimResult(claim.Id, decision.Outcome);
}

[Function(nameof(NudgeLadder))]
public async Task NudgeLadder(
    [OrchestrationTrigger] TaskOrchestrationContext ctx)
{
    var cart = ctx.GetInput<Cart>()!;
    var steps = new[] {
        (TimeSpan.FromHours(1),  "gentle"),
        (TimeSpan.FromHours(24), "discount5"),
        (TimeSpan.FromHours(72), "discount10")
    };

    foreach (var (delay, template) in steps)
    {
        await ctx.CreateTimer(ctx.CurrentUtcDateTime.Add(delay), CancellationToken.None);
        var status = await ctx.CallActivityAsync<CartStatus>("CheckCart", cart.Id);
        if (status.IsCheckedOut || status.IsOptedOut) return;
        await ctx.CallActivityAsync("SendNudge", new NudgeReq(cart, template));
    }
}
