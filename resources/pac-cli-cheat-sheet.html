PAC CLI — Power Platform CLI Cheat Sheet

Latest Version (post v1.34)  |  pac <group> <command> [options]  |  Use pac <command> --help for full options

pac admin Administration & Environments
listList all environments from your tenant
createCreate a new Dataverse instance in your tenant
deleteDelete the environment from your tenant
copyCopy source environment to a destination environment
backupTake a manual backup of the environment
restoreRestore an environment to a given backup
list-backupsList all backups of the environment
resetReset the environment from your tenant
statusList the status of all operations in progress
set-backup-retention-periodSet backup retention in days (valid: 7, 14, 21, 28)
assign-userAssign a user to an environment with a specified security role
assign-groupAssign a group to an environment with a specified security role
list-app-templatesList all supported Dataverse templates of model-driven apps in D365
create-service-principalAdd Entra ID application & application user to the environment
list-service-principalList Entra ID applications which have access to Dataverse
set-governance-configEnable, disable, and edit managed environments
set-runtime-stateUpdate environment administration mode
list-tenant-settingsList tenant-level settings
update-tenant-settingsUpdate tenant-level settings
list-groupsList environment groups from tenant NEW
add-groupAdd an environment to a group NEW
list-rolesList security roles from environment NEW
self-elevateSelf-elevate to System Administrator role NEW
application listList Entra ID applications registered under the tenant
application registerRegister Entra ID application with the tenant
application unregisterUnregister Entra ID application from the tenant
dlp-policy listList DLP policies in the tenant NEW
dlp-policy showShow details of a specific DLP policy NEW
pac auth Authentication Profiles
createCreate and store an authentication profile on this computer
listList the authentication profiles stored on this computer
selectSelect which authentication profile should be active
deleteDelete a particular authentication profile
updateUpdate name or target environment of an existing profile
nameName or rename an existing authentication profile
clearClear all authentication profiles stored on this computer
whoDisplay information about the currently selected auth profile
pac env Dataverse Organization (replaces pac org)
listList all Dataverse environments the user has access to
whoDisplay information about the current Dataverse organization
selectSelect default org for current authentication profile
fetchPerform FetchXML query against Dataverse
list-settingsList environment settings
update-settingsUpdate environment settings
💡 pac org is still supported as an alias
pac solution Solution Project Management
initInitialize a directory with a new Dataverse solution project
add-referenceAdd a reference from the project in the current directory to the project at 'path'
listList all solutions from the current Dataverse organization
deleteDelete a solution from Dataverse in the current environment
exportExport a solution from Dataverse
importImport the solution into Dataverse
cloneCreate a solution project based on an existing solution in your org
publishPublish all customizations
upgradeApply solution upgrade
versionUpdate build or revision version for the solution
online-versionSet version for solution loaded in Dataverse
packPackage solution components on local filesystem into solution.zip (SolutionPackager)
unpackExtract solution components from solution.zip onto local filesystem (SolutionPackager)
checkUpload a Dataverse solution project to run against the Power Apps Checker service
create-settingsCreate a settings file from solution zip or solution folder
add-solution-componentAdd one or more solution components to the target unmanaged solution
add-licenseAdd license and plan info to the solution
syncSync the current solution project to the current state of the solution in your org
pac canvas Canvas Apps (.msapp)
listList canvas apps in the environment
downloadDownload canvas app as .msapp file
createGenerate a canvas app from a custom connector
packPack sources into an msapp file Preview
unpackExtract an msapp file into YAML sources Preview
validateValidate the .pa.yaml source for an unzipped msapp file Preview
pac pcf Power Apps Component Framework
initInitialize a directory with a new PCF project
pushImport the PCF project into the current Dataverse organization
versionPatch version for controls
pac plugin Dataverse Plug-ins
initInitialize a directory with a new Dataverse plug-in class library
pushImport plug-in into Dataverse
pac connection Dataverse Connections
createCreate a new Dataverse connection
deleteDelete a Dataverse connection by connection-id
listList all connections (shows Id, Name, API Id, Status)
updateUpdate a Dataverse connection
💡 --json not supported on list; parse Status column with PowerShell
pac connector Custom Connectors
listList the connectors registered in Dataverse
initInitialize a new API Properties file for a connector
createCreate a new row in the Connector table in Dataverse
downloadDownload a Connector's OpenAPI definition & API Properties file
updateUpdate a Connector entity in Dataverse
pac copilot Copilot / Bot Management
listList copilots in the current or target environment
statusPoll deployment status of a specified copilot
createCreate a new copilot using an existing template file
publishPublish a custom copilot
extract-templateExtract a template file from an existing copilot
extract-translationExtract file containing localized content for bots
merge-translationMerge files containing localized content for bots
init-skillsExtract copilot skill definition files to current directory NEW
mcpInfo about local MCP (Model Context Protocol) server NEW
model listList AI Builder models in current environment NEW
model predictSend text/prompt to an AI Model NEW
pac pipeline Deployment Pipelines
listList all pipelines in the environment
deployStart a pipeline deployment
pac data Import / Export Data
exportExport data (uses Configuration Migration tool)
importImport data (uses Configuration Migration tool)
⚠ .NET Full Framework version only
pac package Package Deployer
initInitialize a directory with a new Dataverse package project
add-external-packageAdd package external to Dataverse solution system to a Package Deployer project
add-solutionAdd prebuilt Dataverse solution file to a Package Deployer project
add-referenceAdd reference to Dataverse solution project
deployDeploy package to Dataverse (.NET Full Framework only)
showShow details of Dataverse package (.NET Full Framework only)
pac pages Power Pages (replaces pac powerpages)
listList all Power Pages websites from the current environment
downloadDownload Power Pages website content from the environment
uploadUpload Power Pages website content to the current environment
migrate-datamodelManage data model migration for your Power Pages website
bootstrap-migrateMigrate HTML code from Bootstrap V3 to V5
cloneCreate website content from existing content NEW
download-code-siteDownload Power Pages website content from environment NEW
upload-code-siteUpload compiled code to Power Pages site NEW
💡 pac paportal and pac powerpages still work as aliases

prtLaunch Plug-in Registration Tool (PRT)

pdLaunch Package Deployer (PD)
