üìù Method 1: JavaScript Functions

Add components dynamically using JavaScript functions:

JavaScript Component Usage
javascript 
// Define related articles
const relatedArticles = [
    {
        url: 'article1.html',
        title: 'First Related Article',
        description: 'Description of the first article',
        disabled: false
    },
    {
        url: '#',
        title: 'Coming Soon Article',
        description: 'This article is not ready yet',
        disabled: true
    }
];

// Add related articles to page
BlogComponents.addRelatedArticles(relatedArticles);

// Add share section with custom text
BlogComponents.addShareSection(
    'üöÄ Share This Demo', 
    'Show your team how components work!'
);

‚öôÔ∏è Configuration Options

Article Object

Related Article Structure

  • url: Link to the article (use '#' for disabled)
  • title: Article title displayed on card
  • description: Brief description of the article
  • disabled: Boolean to show as "coming soon"
üí°

Best Practices

  • Limit related articles to 2-4 items for better readability
  • Use descriptive titles and concise descriptions
  • Set disabled: true for articles that aren't ready yet
  • Customize share section text to match your content
